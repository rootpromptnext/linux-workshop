Linux Workshop

This workshop is designed to help learners build strong Linux foundations with practical, hands-on labs. You will work directly on a Linux machine and practice commands used every day by system administrators and DevOps engineers.

Objective

Learn Linux commands, file management, permissions, processes, networking basics, and shell scripting through real-time labs.

Audience

  • Beginners wanting to learn Linux
  • IT professionals wanting command-line skills
  • Anyone preparing for DevOps or cloud roles

Prerequisites

  • Basic computer knowledge
  • Linux VM (Ubuntu/CentOS) or cloud instance
  • Terminal or SSH access

Workshop Structure (6 Modules)

  • 1. Linux Basics — Terminal navigation, file system, OS and kernel info
  • 2. File Management — Create, copy, move, delete, search files
  • 3. Users, Groups & Permissions — User management, chmod, chown, sudo
  • 4. Processes & Services — ps, top, systemctl, service handling
  • 5. Networking Basics — IP, ping, traceroute, ports, SSH
  • 6. Shell Scripting Basics — Variables, loops, conditions, scripts

By completing this Linux workshop, you will gain the essential skills needed to comfortably work on servers and troubleshoot real-time issues with confidence.
